# 👕 Zoich - Clothing Brand Website

A modern and responsive clothing brand website currently under development.

This project started as a homepage prototype built using HTML and CSS and will gradually evolve into a complete Full-Stack E-Commerce Application using Java, Spring Boot, and MySQL.

# 📖 About the Project

Zoich is a clothing brand website designed with a clean, premium, and modern user interface.

The goal is to build a real-world full-stack e-commerce application where users can browse products, manage their cart, place orders, and securely log in.

This repository currently contains the homepage UI and will be updated as new features are developed.
